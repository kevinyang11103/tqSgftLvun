# 学生选课管理系统 python497

## 项目概述

学生选课管理系统是基于 Python Flask 框架和 MySQL 数据库开发的一套在线系统。该系统旨在简化学生选课流程，提高课程管理效率。

## 技术栈

- 后端：Python Flask
- 数据库：MySQL

## 功能模块

### 系统角色

- 学生
- 教师
- 管理员

### 核心功能

1. **登录功能**：各角色用户均可登录系统。
2. **学院管理**：管理学院的增删改查操作。
3. **班级管理**：对班级信息进行管理。
4. **专业信息管理**：维护专业相关资料。
5. **课程信息管理**：包括课程的增加、修改、查询和删除。
6. **学生信息管理**：管理学生资料。
7. **教师信息管理**：维护教师相关数据。
8. **选课信息管理**：处理学生选课请求，记录选课结果。

## 运行环境

- Python 3.x
- Flask 1.x
- MySQL 5.x

## 目录结构

```plaintext
/
├── app                 # Flask 应用目录
│   ├── __init__.py
│   ├── models.py       # 数据模型定义
│   ├── views.py        # 视图处理逻辑
│   └── templates       # HTML 模板目录
│       └── ...
├── migrations          # 数据库迁移目录
├── venv                # 虚拟环境目录
├── requirements.txt    # 依赖列表
└── run.py              # 应用启动脚本
```

## 核心亮点

- **简洁的界面**：系统界面简洁，易于操作。
- **灵活的角色权限控制**：不同的角色拥有不同的操作权限。
- **高效的数据管理**：使用关系型数据库 MySQL，保证数据的一致性和安全性。

## 部署步骤

1. 安装 Python 3.x 和 MySQL。
2. 创建数据库和用户，配置数据库连接。
3. 安装依赖（`pip install -r requirements.txt`）。
4. 运行数据库迁移（`flask db init`、`flask db migrate`、`flask db upgrade`）。
5. 启动 Flask 应用（`python run.py`）。
